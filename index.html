Cronograma de Desarrollo

12 semanas de desarrollo estructurado y entrega incremental

1

Análisis de Requerimientos

Semana 1

Reunión con gerencia y equipo de EGISPRO. Levantamiento de procesos actuales y flujos con las ARL. Definición de roles de usuario (admin, profesional, cliente). Documentación inicial de alcance.

Entregable: Documento de requerimientos y alcance del proyecto
2

Diseño de Arquitectura

Semana 2

Diagramas de arquitectura (frontend, backend, base de datos, integraciones externas). Selección de stack definitivo (Next.js, Node, Prisma, PostgreSQL, Azure). Configuración de repositorios y entorno de desarrollo.

Entregable: Arquitectura documentada y entorno configurado
3

Modelado de Base de Datos

Semana 3

Diseño de entidades (Órdenes, Profesionales, Clientes, Actividades, Documentos). Creación de esquema inicial con Prisma + PostgreSQL. Migraciones iniciales y validación de relaciones.

Entregable: Base de datos modelada y configurada
4

Mockups y UX/UI

Semana 4

Diseño de pantallas principales: dashboard, órdenes, calendario. Prototipo visual en Figma o similar. Validación de interfaz con gerencia.

Entregable: Diseños UI/UX aprobados y sistema de diseño
5

Backend Inicial

Semana 5

Configuración de API GraphQL/REST. Endpoints base para Órdenes y Profesionales. Autenticación JWT y roles.

Entregable: API base funcional con autenticación
6

BOT AI (Scraping ARL)

Semana 6

Desarrollo del bot (Puppeteer/Playwright). Automatización de login y consulta de órdenes. Registro automático en la base de datos.

Entregable: BOT inteligente funcional y monitoreado
7

OCR y Documentos

Semana 7

Integración de Azure Cognitive Services (OCR). Extracción de datos de documentos de soporte. Validación de estructura de archivos y almacenamiento en Azure.

Entregable: Sistema OCR integrado con Azure
8

Notificaciones

Semana 8

Integración con Outlook SMTP para correos. Integración con API de WhatsApp Business. Sistema de recordatorios automáticos.

Entregable: Sistema de notificaciones completo
9

Calendario y Profesionales

Semana 9

Integración con Google/Outlook Calendar API. Módulo de agenda de profesionales. Vista mensual con actividades programadas.

Entregable: Módulo de profesionales y calendarios integrados
10

Dashboard y Reportes

Semana 10

Dashboard central con métricas de estado de servicios. Exportación de reportes en PDF y Excel. Gráficas con estadísticas de pérdidas/ahorros.

Entregable: Dashboard completo y sistema de reportes
11

Pruebas y QA

Semana 11

Pruebas unitarias y de integración. Ajustes de seguridad (tokens, permisos). Corrección de bugs.

Entregable: Sistema probado y auditado
12

Despliegue y Capacitación

Semana 12

Deploy final en Vercel + Azure. Entrenamiento a personal de EGISPRO. Entrega de manual de usuario y documentación técnica.

Entregable: Plataforma en producción + Capacitación completa
