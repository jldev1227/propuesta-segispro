Semana 1

Definición de requerimientos y alcance

Reunión con el equipo de SEGISPRO para levantar información, entender los flujos con las ARL y definir roles, permisos y objetivos del sistema.

🎯 Entregable

Documento de requerimientos y alcance aprobado por el equipo.

Semana 2

Diseño de la arquitectura

Creación de diagramas técnicos de arquitectura (frontend, backend, base de datos y APIs). Configuración de entorno en Azure y repositorios.

🎯 Entregable

Diagramas de arquitectura aprobados y entorno de desarrollo configurado en Azure.

Semana 3

Modelado de base de datos

Diseño del modelo relacional con Prisma y PostgreSQL. Creación de migraciones y validación de relaciones entre órdenes, clientes y profesionales.

🎯 Entregable

Modelo de base de datos aprobado, migraciones creadas y relaciones validadas entre entidades principales.

Semana 4

Diseño UX/UI

Creación de prototipos interactivos en Figma, definición de componentes reutilizables, paleta de colores y diseño responsive del panel de control principal.

🎯 Entregable

Prototipo visual aprobado y flujo de navegación definido.

Semana 5

Desarrollo backend inicial

Creación de endpoints base para gestión de órdenes, autenticación JWT y roles de usuario. Conexión con la base de datos y pruebas iniciales de la API REST.

🎯 Entregable

API funcional con endpoints para órdenes, autenticación y roles, conectada a la base de datos.

Semana 6

BOT AI (automatización ARL)

Implementación de un bot inteligente con Puppeteer que accede automáticamente a portales ARL, consulta órdenes y las registra en la base de datos. Incluye manejo de autenticación, navegación dinámica y extracción robusta de datos.

🎯 Entregable

Bot funcional que automatiza la consulta y registro de órdenes ARL, con logs de actividad y validación de resultados en la base de datos.

Semana 7

OCR y documentos

Integración avanzada con Azure Cognitive Services para lectura OCR de soportes. Automatización del almacenamiento, validación y extracción inteligente de datos desde documentos escaneados.

🎯 Entregable

Módulo funcional de OCR con Azure, capaz de leer, validar y almacenar documentos automáticamente en la plataforma.

Semana 8

Notificaciones automáticas

Implementación de envío de correos con Outlook SMTP y recordatorios vía WhatsApp API para aceptación y seguimiento de órdenes. Configuración de alertas inteligentes y programación de actividades.

🎯 Entregable

Módulo funcional de notificaciones automáticas por correo y WhatsApp, con seguimiento y confirmación de programación o reprogramación.

Semana 9

Calendario y profesionales

Integración avanzada de calendarios Outlook y Google. Desarrollo del módulo de agenda para visualizar actividades, asignaciones y disponibilidad de cada profesional en tiempo real.

🎯 Entregable

Módulo funcional de calendario integrado y gestión de profesionales, con visualización de agenda y asignaciones.

Semana 10

Administración y reportes

Desarrollo del panel administrativo con métricas de desempeño, estado de órdenes y reportes exportables en Excel y PDF. Visualización interactiva y filtros avanzados para análisis de datos.

🎯 Entregable

Panel funcional con métricas, exportación de reportes y visualización de estado de órdenes en tiempo real.

Semana 11

Pruebas y control de calidad

Pruebas unitarias e integrales del sistema. Validación de seguridad, rendimiento y estabilidad antes del despliegue final.

🎯 Entregable

Sistema validado y listo para despliegue, con pruebas de calidad y seguridad superadas.

Semana 12

Despliegue y capacitación

Deploy final en Azure y Vercel, documentación técnica, entrega oficial y capacitación al equipo interno de SEGISPRO.

🎯 Entregable

Plataforma desplegada, documentación entregada y equipo capacitado para operación y soporte.
